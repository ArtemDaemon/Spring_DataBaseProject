## Getting Started

### Installation

1. Clone the repository to your local machine:
```
git clone https://github.com/ArtemDaemon/Spring_DataBaseProject
```
2. Navigate to the project directory:
```
cd <project-directory>
```
3. Build the project using Maven:
```
mvn clean install
```
